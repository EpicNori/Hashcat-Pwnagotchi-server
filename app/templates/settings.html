How do these limits work?
  • Dual GPU Support: You can now specify individual intensity for up to 2 GPUs.
    • 0%: Disables the device completely.
    • 1-100%: Enables the device and sets the global workload profile (-w) based on the highest setting.
  • Workload Mapping (-w):
    • 1-25%: Low usage (Desktop remains responsive)
    • 26-50%: Medium (Default balanced mode)
    • 51-75%: High (Intense cracking)
    • 76-100%: Nightmare (Maximum heat/Full utilization)
  • Cracking Strategy:
    • Wordlists: By default, the server uses optimized lists of proven passwords.
    • Rainbow: Replays the rainbow reuse list built from previously cracked keys.
    • Every Number: To try every possible 8-digit number, simply select the digits_8.txt wordlist on the upload page.
