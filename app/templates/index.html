Hashcat WPA/WPA2 Server

Hashcat WPA/WPA2 Server is a web-based platform for managing WPA and WPA2 handshake cracking jobs from one central dashboard. It helps organize uploads, monitor active hardware usage, and review successful or failed task results in real time. The system is built to make Hashcat workflows easier to manage across CPU and GPU devices.
